# DICOM Medical Viewer & Integration Tool

A web-based medical imaging tool built to work with DICOM data, combining a responsive frontend UI with backend services for handling and serving medical images.  
The project is designed to support radiology and healthcare workflows through browser-based visualization and system integration.
